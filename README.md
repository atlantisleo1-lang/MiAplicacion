# Dashboard Analítico con Streamlit - Fundamentos de Programación

Este repositorio contiene el proyecto final del módulo "Especialización en Python for Analytics". Es una aplicación web interactiva desarrollada con Python y Streamlit que automatiza procesos matemáticos, financieros y de gestión de inventarios.

## 🚀 Enlace al Proyecto
Puedes probar la aplicación en vivo aquí: https://miaplicacion-le.streamlit.app/

## 🛠️ Tecnologías Utilizadas
* **Python 3** (Lógica base y Programación Orientada a Objetos)
* **Streamlit** (Interfaz gráfica y gestión del `session_state`)
* **Pandas** (Estructuración de datos en DataFrames)
* **NumPy** (Procesamiento de arreglos)

## 📋 Módulos de la Aplicación
La aplicación está dividida en 4 ejercicios prácticos:
1. **Flujo de Caja:** Sistema CRUD en memoria para registrar ingresos y gastos diarios, calculando el saldo final automáticamente.
2. **Registro de Productos:** Formulario que utiliza arreglos de NumPy para almacenar productos, categorías y calcular precios totales.
3. **Indicadores de Mantenimiento:** Calculadora técnica para obtener KPIs industriales como el MTBF (Tiempo Medio Entre Fallas), MTTR y Disponibilidad, guardando un historial mensual.
4. **Inventario Inteligente (POO):** Sistema de gestión de stock utilizando Clases y Objetos en Python para empaquetar y analizar la información de los productos.
